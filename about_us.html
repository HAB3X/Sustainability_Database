About the Sustainability Database

    This project is a free, open-source directory of businesses and sectors across Australia and New Zealand with a focus on sustainability, governance and social impact.

    Entries are grouped into three major sectors: Fashion, Infrastructure & Industry, and Finance. Each profile summarises what the company does, where it operates, and how it performs on key environmental, social and governance indicators.

    All content is backed by public sources and documented in a transparent way using GitHub, Markdown files and structured data.

    How the directory is built

    • Profiles draw on company reports, regulatory filings, credible news coverage and independent databases.
    • Information is structured consistently to help users compare companies and sectors.
    • Research notes, issues and updates are managed via GitHub for transparency.
